# HS200 Device-Specific Planning

This document outlines the device-specific planning for the HS200 smart switches.

## Devices

*   **hs200-1**
*   **hs200-2**

## Resources

These are simple IoT devices and do not require significant resource planning (CPU, RAM, storage).

## Configuration

*   **Firmware:** Stock firmware
*   **VLAN:** IoT VLAN
*   **Firewall Rules:** Allow access to Home Assistant and local network for management. Block internet access.
